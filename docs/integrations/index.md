# Integrations

IncidentRelay supports alert intake from:

- Alertmanager;
- Zabbix;
- generic webhooks.

It can send notifications to:

- Mattermost;
- Slack;
- Telegram;
- Discord;
- Teams;
- email;
- webhook;
- voice call providers.
